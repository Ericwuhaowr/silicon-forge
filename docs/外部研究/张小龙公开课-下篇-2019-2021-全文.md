## 2019年：微信8周年，DAU破10亿

### 设计原则：微信的"异类"哲学

**十大设计原则（源自德国设计师Rams）：**
好的产品是富有创意的、有用的、美的、容易使用的、含蓄不招摇的、诚实的、经久不衰、不放过任何细节、环保的、尽可能少的设计（少即是多）。

> "微信只是守住了做一个好产品的底线，居然就与众不同了。那是因为很多产品不把自己当产品看待，不把用户当用户看待。"

> "我怎么舍得在你最好的朋友脸上，贴一个广告呢？"

### 微信的两大"原动力"

| 原动力 | 核心内涵 |
|--------|----------|
| 第一：坚持做一个好的、与时俱进的工具 | 微信是一个陪伴用户多年的老朋友，不是榨取用户价值的机器 |
| 第二：让创造者体现价值 | 打破信息不对称，让再小的个体也有自己的品牌 |

### 关于"停留时长"的反常识洞察

> "技术的使命应该是帮助人类提高效率。"

- 微信永远不会把用户停留时长作为目标
- 朋友圈每天人均时长基本固定（约30分钟），不随内容量增加而增加
- 已发送状态、已读回执等被刻意不做的原因：最高效率就是"发完即走"

### 小程序：微信史上最大挑战

**2019年小程序三大重点方向：**
1. 搜索直达 — 用户可直接搜索到小程序内部内容
2. 完善的评价体系 — 通过社交评价建立信任
3. 小程序找回 — 通过强提醒等能力增强用户粘性（拒绝粗暴推送）

> "只让自己好不让别人好的事情，通常都不会太长久。"

### 小游戏：关于创意的平台

小游戏的原动力不是公司盈利渠道，而是关于创意的平台，让产生创意的人体现价值。

> "我不希望只是看到我们的收入又上涨了多少，反而我希望看到的是这里特别多的游戏是一些从来没有做过游戏的人做的。"

### 关于社交的本质

**张小龙对社交起源的想象：**
远古时代，人在社区里最大的诉求是不被排斥。人必须通过"吹牛"来体现自己对社区的价值。朋友圈的本质就是延续这种习惯。

> "沟通就是把自己的人设强加给对方的过程。"

**朋友圈的本质：一个"广场"**
- 朋友圈开创了新的社交场所，不止是时间流
- 用户每天花30分钟"逛广场"，完成一天的社交任务
- 7.5亿人每天进入朋友圈，总量达100亿次

**朋友圈的压力与"三天可见"：**
超过1亿人设置了三天可见，是微信里最多人用的开关。视频动态定位为"朋友圈的反面"，提倡真实而非美化。

> "朋友圈是展现自己最美好的一面，视频动态是展现自己最真实的一面。"

### 关于阅读与"看一看"

张小龙坚信社交推荐是最符合人性的信息获取方式。人类社交体系具有"纠错功能"，机器推荐容易强化惰性。

> "本质上我们并不是在做阅读，而是每一个人在帮助其他人阅读。"

### 关于AI的警惕

> "当AI很强大以后，AI可能是可以驾驭人的。"

传统工具是"人驾驭工具"，但AI可能反过来成为"驾驭人的工具"。对AI在微信中的应用保持理性警惕。

### 关于"善良"

> "善良本质上是一种能力。是一种基于理性之上的善良。"

对用户的态度必须是善良，而非套路。微信团队从开始到现在从未瞄准KPI奋斗。

### 关于企业微信

企业微信的新理念：让每个企业员工成为企业服务的窗口。

> "人就是服务，而且是认证的服务。"

员工加客户微信后，对话界面直接带服务菜单。人比APP/小程序更有灵活性。

### 关于微信支付

- 支付做得好到"不需要提到"
- 红包新功能：企业微信可申请红包封皮
- 亲属卡：被特别推荐的"藏得太深"的好功能

### 结尾金句

> "万物之中，希望至美。"

## 2020年：张小龙首次缺席现场，视频演讲

### 信息互联的七大影响

| 影响维度 | 核心观点 |
|----------|----------|
| 隐私的出让 | 科技越发达，个人隐私越少；精准广告与隐私保护存在矛盾 |
| 信息获取的被动 | 推送决定了用户看到什么世界；少说"分发"，多用"推荐" |
| 社会关系的扩大 | 邓巴数（150个好友）已被打破；近百万人接近5000好友上限 |
| 信息传播的快速 | 谣言传千里是人性使然；需用更多参与者和机制来仲裁 |
| 信息选择的困难 | 社交推荐通过好友互相推荐扩大选择范围 |
| 信息的多样性 | 长尾小号需要生存空间 |
| 搜索的困难 | 小程序支撑长尾搜索需求，搜索需进入每个小程序内部 |

### 两个"小小失误"

1. 公众平台长期只有PC Web版 — 限制了内容创作者范围
2. 公众号被做成了文章载体 — 短内容形式缺失，普通人的创作入口被忽略

> "我们缺少了一个人人可以创作的载体。因为不能要求每个人都能天天写文章。"

## 2021年：微信十周年，全新产品矩阵

### 微信十年数据

| 指标 | 每日数据 |
|------|----------|
| 打开微信 | 10.9亿人 |
| 视频通话 | 3.3亿人 |
| 进入朋友圈 | 7.8亿人 |
| 发表朋友圈 | 1.2亿人（照片6.7亿张，视频1亿条） |
| 阅读公众号文章 | 3.6亿人 |
| 使用小程序 | 4亿用户 |

### 视频号的诞生与进化

**起源：** 2017年提出"非朋友圈"概念，2019年组建小团队（约一二十人）开始开发

**定位演变：**
- 最初定位："视频化的微博"
- 重新定义："人人都可以创作的短内容平台，公开领域的内容平台"

**视频号ID的关键价值：**
- 微信身份（ID）是私密领域的，视频号ID是公开领域的
- 有了视频号ID，直播才真正变成公开领域的事情
- "视频号才是你真正的官网"

**内容消费比例预测（1:2:10）：**
- 关注内容：1
- 朋友点赞：2
- 机器推荐：10
- 实际数据验证为1:2:9

**视频的重新定义：**
> "视频文件这种东西以后可能会消失。云端化、结构化的视频才是视频，本地化的视频文件都是裸视频，应该被淘汰掉。"

**全屏化逻辑：**
> "一屏里面的内容条数，和这页里用户感兴趣的命中率成反比。"

### 直播的未来判断

> "未来直播会变成一种很多人都在用的个人表达方式。"

- 直播比短视频创作门槛更低，更轻松
- 终极形态：点开名片看到对方眼镜里的直播画面
- "每个人都是别人的眼睛，每个人都可以通过别人的眼睛看到他所看到的"

### 创作者生态

> "我们不花钱买内容，创作者还愿意进来，这才说明可以在这里根据自己的内容获得回报。"

不签约明星、不购买内容。依靠系统和规则，而非运营驱动。微信支付每个行业只配一两个人。

### 新产品实验

| 功能 | 说明 |
|------|------|
| 新表情 | 炸弹、烟花等强烈情绪表达 |
| 个人状态 | 随便写几个字表达状态，找到"同类" |
| 听歌可视化 | 听歌时看到画面 |
| 新浮窗 | "比优雅10倍的方案" |
| 微信输入法 | 基于隐私保护，回应聊天记录被监听投诉 |

### 微信十年总结

> "归结为非常简单的一两个词来表达的话，我想就两个词，一个是连接，一个是简单。"

> "一个产品要加多少功能才能不是一个垃圾产品。"

> "我还是希望它一直保持自己的风格，一直像一个小而美的产品一样，有自己的灵魂和审美，有自己的创意和观念，而不仅仅是一个数字目标的奴隶。"
