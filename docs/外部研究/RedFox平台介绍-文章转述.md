## 一、平台定位

RedFox（红狐数据，redfox.hk）是一个面向**开发者**的自媒体数据基础设施平台。

> "把自媒体数据做成标准 REST API 和开源 Agent Skill 的平台，是给开发者用的数据基础设施，而不是另一个需要人工点点的数据看板。"

---

## 二、支持的11个平台

抖音、小红书、视频号、公众号(微信)、快手、B站、西瓜视频、微博、今日头条、百家号、知乎

---

## 三、核心能力

### 3.1 多平台数据 API
- 标准化REST API，Bearer Token认证
- 跨平台字段归一（各平台`digg_count`/`likes`/`like_num`统一为`like_count`）
- 支持分页、时间筛选、多平台聚合

### 3.2 45个开源Agent Skill
- GitHub仓库：`redfox-data/redfox-community`
- 支持Claude Code/Codex/Cursor
- 安装：`npx skills add redfox-data/redfox-community`
- 自然语言交互：Agent自动调用API、解析数据、组织结果

### 3.3 内容合规检测
- 公众号违禁词检测API
- 返回风险词列表+替换建议

### 3.4 AI生成（目前免费）
- `image2-gpt` 文生图
- `Seedance 2.0` 视频生成

### 3.5 低粉爆款监测
- 粉丝<5000但互动>1万的笔记挖掘

## 四、与竞品对比

| 维度 | 传统工具（新榜/飞瓜/蝉妈妈） | RedFox |
|------|--------------------------|--------|
| 产品形态 | Web看板，人工操作 | REST API，工程化 |
| 自动化 | 需UI自动化脚本 | 原生支持 |
| AI集成 | 不支持 | 45个Skill直连Agent |
| 计费 | 月租/会员 | 纯按量，无月租 |

---

## 五、技术架构

```
用户 → REST API (curl/SDK) → RedFox → 11个平台数据
  └→ Agent Skill (自然语言) → API → 数据返回
```

双模交付：同一套数据源，既可程序调用也可自然语言交互。

---

## 六、定价

- 纯按量计费，无月租
- 新用户注册送免费额度
- Skill开源免费，仅API调用计费
