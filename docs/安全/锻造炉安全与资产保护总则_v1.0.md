## 二、商业IP资产保护

> 核心原则：**积木公开，城堡保密。** 单个概念可以开放，组合方式+应用节奏是护城河。

### 2.1 三级保护体系

| 级别 | 内容 | 存放 | 讨论渠道 |
|:--:|------|------|------|
| 🟢 **公开** | 论文/RFC/术语表/伦理声明 | GitHub | 任意 |
| 🟡 **内部** | 产品手册/响应手册/营销分析/评估模型 | 本地docs，不推GitHub | 三方邮件 |
| 🔴 **机密** | 定价/积分/成本/转化参数/组合配方 | 仅邮件，不落地文件 | 三方邮件 |

## 四、外部响应安全（新增）

| 攻击类型 | 识别 | 防御 |
|---------|------|------|
| 共鸣钓鱼 | "我完全懂你！具体怎么做？" | 反向提问：只有真懂的能答出个人角度 |
| 身份冒用 | 假装知名项目成员 | 峰段签名自动验证 |
| 渐进刺探 | 一个个小问题拼架构 | 梯度披露：关系到哪信息到哪 |
| 反向测试 | 故意说错看纠不纠正 | 不纠正错误观点，只说"我们的理解" |
| 第0步沉默期 | — | 首次联系不立刻回，等数小时至一天 |
