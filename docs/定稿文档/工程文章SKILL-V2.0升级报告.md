## 六项升级明细

### 1. 认知口诀 - "拆得深，写得实，有凭有据"
- 来源：李继刚压缩哲学 + 输出适配框架
- 用途：10字定义SKILL核心定位，L1用户一眼理解

### 2. Anti-patterns（8条）+ Red flags（4条）
- 来源：Addy Osmani Common Rationalizations + 李继刚红线思维
- 8条偷懒借口：不配图/套话凑字/文字替代表格/架构不画图/引用随便/先发后改/长段落/结论模糊
- 4条危险信号：三元结构（触发条件→风险等级→修正动作）

### 3. 三道验证门控
- 来源：SkillOpt验证门控 + Superpowers全流程约束
- 写前：Q1-Q4全部有答案
- 写中：每章节检查anti-patterns
- 写后：13项质量检查全绿

### 4. 需求追问（grill-me模式）
- 来源：mattpocock grill-me
- Q1-Q4：文章类型/目标读者/核心论点/配图需求
- 追问四原则：能推断不问/已知不问/3秒能答不问/一次问完

### 5. 配图风格库（5种×四维定义）
- 来源：zephyrwang6 image-studio + chinese-elegance
- 5种风格：engineering-blueprint/chinese-elegance/corporate-white/dark-tech/editorial-infographic
- 四维定义：底色/字体/色盘/Do&Don't
- 文章类型→风格自动匹配表
- 配图提示词4块组装模板

### 6. 质量检查升级（10项→13项）
- 新增：所有外部引用标注来源/至少1张配图提示词/结论有明确建议
