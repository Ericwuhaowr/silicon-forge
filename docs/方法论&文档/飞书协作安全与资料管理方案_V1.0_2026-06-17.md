## 二、飞书文档/云盘安全管理

### 2.1 权限分级

| 文档级别 | 可见范围 | 编辑权 | 示例 |

| 🔴 核心 | 仅三方 | 吴昊+马斯克 | SKILL框架、gotchas库、巡检源清单内部版 |
| 🟡 内部 | 仅三方 | 三方均可 | 巡检报告、评估意见、方法论草稿 |
| 🟢 可公开 | 三方 + 可降至仅查看 | 三方 | 最终发布版方法论、可公开的避坑经验 |

### 2.2 操作规范

| 规则 | 说明 |
|------|------|
| **默认仅三方可见** | 新文档/文件夹创建时默认不公开分享，飞书默认设置"仅协作者" |
| **敏感文档加密** | 包含 🔴 级别内容的飞书文档，导出/分享前必须脱敏 |
| **外发脱敏** | 发给第三方的文档必须去掉真实数据、具体人名、内部指标 |
| **定期权限审查** | 每月检查一次飞书文档的协作者列表，移除不再需要的授权 |
| **历史版本保留** | 利用飞书文档自动版本历史作为审计追溯 |

## 四、协作流程中的安全检查点

| 阶段 | 检查项 |
|------|------|
| **任务启动** | 确认数据级别（🔴/🟡/🟢），选择对应的共享渠道 |
| **文档共享** | 飞书文档权限设置为"仅协作者"，不公开发链接 |
| **文件上传** | 文件上传前确认无凭证、无客户数据 |
| **交付前** | 🟡 级及以上文档发出前做脱敏检查 |
| **关项** | 确认所有敏感中间产物已归档或清理 |

## 五、应急响应

| 情况 | 响应 |
|------|------|
| **飞书账号疑似泄露** | 立即修改飞书密码 + 重设授权token + 检查最近登录设备 |
| **文档误公开** | 立即撤回公开链接 + 改为仅协作者 + 追溯访问记录 |
| **群聊进了不明人员** | 立即踢出 + 检查邀请记录 + 重设群分享链接 |
| **凭证在群聊中误发** | 立即删除消息 + 三个平台各自重置相关凭证 |
